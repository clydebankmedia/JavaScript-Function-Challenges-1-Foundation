JavaScript Function Challenges - The Scholar's Path

Instructions: Open this project folder in your Visual Studio Code editor. Then open script.js and write your solution directly in that file. Save your changes, then open index.html in your web browser to test your code.

You can view your output in the browser’s Developer Console. To open it in Google Chrome browser:
– On Windows: press Ctrl + Shift + I and select the Console tab
– On Mac: press Cmd + Option + I and select the Console tab

Steps

  1. Open index.html in your browser.
  2. Follow the directions in the QuickStart Academy project page.
  3. Edit and save your code in script.js.
  4. Check the Console output for results or errors.
  5. When finished, return to the learning path and choose the next room challenge.
